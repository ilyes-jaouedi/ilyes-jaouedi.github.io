# ilyes-jaouedi.github.io

Personal academic portfolio of **Ilyes Jaouedi** — PhD Candidate at the [Laboratory of Signals and Systems (L2S)](https://l2s.centralesupelec.fr/en/), CentraleSupélec / CNRS / Université Paris-Saclay.

**Live site:** [ilyes-jaouedi.github.io](https://ilyes-jaouedi.github.io)

## Research

Camera-radar fusion via optimal transport for Advanced Driver Assistance Systems (ADAS), in industrial collaboration with [FORVIA](https://www.forvia.com).

## Stack

- [Next.js](https://nextjs.org/) — React framework (static export)
- [Tailwind CSS](https://tailwindcss.com/) — styling
- [Framer Motion](https://www.framer.com/motion/) — animations
- Deployed via GitHub Actions → GitHub Pages

## Publications

The publications section fetches live from [HAL](https://hal.science) and updates automatically.
